## 🚀 Следующие шаги

```

   ├─ Скопировать .env.example в .env
   ├─ Получить Telegram Bot Token
   ├─ Получить OpenAI API Key
   └─ python init_project.py

2. TEST (сегодня)
   ├─ python test_oracle.py
   └─ Протестировать в Telegram
```
