## ✨ Новые модули

### 1. 📊 Натальная карта (`/natal`)

**Что это?**  
Полная астрологическая карта рождения с расположением планет, домов и аспектов.

**Что нужно:**
- Дата рождения (дд.мм.гггг)
- Время рождения (чч:мм)
- Город рождения (или координаты)

**Что получите:**
- ☀️ Солнце, Луна, Асцендент
- 🌍 Баланс элементов (огонь, земля, воздух, вода)
- 📍 Положение всех планет в знаках и домах
- 🔗 Основные аспекты между планетами
- 📊 Тип натальной карты (Bundle, Bowl, Bucket)

**Пример:**
```
/natal
15.03.1990 14:30 Москва
```

---

### 2. 🔢 Китайская нумерология Сюцай (`/numerology`)

**Что это?**  
Древняя китайская система расчета чисел судьбы по дате рождения.

**Что нужно:**
- Дата рождения (дд.мм.гггг)

**Что получите:**
- 🎯 Число Жизненного Пути (главное число)
- 💫 Число Судьбы  
- 🧘 Число Души
- 👤 Число Личности
- 🌟 Китайский элемент (Вода, Огонь, Земля, Металл, Дерево)
- ⚡ Энергия Инь или Ян
- 🍀 Счастливые числа, цвета и направления

**Пример:**
```
/numerology
15.03.1990
```

---

### 3. 🔮 Матрица Судьбы (`/matrix`)

**Что это?**  
Система расчета на основе 22 Арканов Таро, популярная в русскоязычной эзотерике.

**Что нужно:**
- Дата рождения (дд.мм.гггг)

**Что получите:**
- 🌟 Личный, Социальный, Духовный Арканы
- 🧘 Чакровая линия (7 чакр с энергиями)
- 💰 Денежный канал
- 🏥 Канал здоровья
- 📋 Кармические программы (родительская, любовная, таланты)
- ⏰ Энергия текущего года
- 🎯 Ваше предназначение

**Пример:**
```
/matrix
15.03.1990
```

---

### 4. ⭐ Ежедневный гороскоп (`/horoscope`)

**Что это?**  
Прогноз на день для вашего знака зодиака.

**Что нужно:**
- Выбрать знак зодиака из меню

**Что получите:**
- 📅 Общий прогноз на сегодня
- 💕 Любовь
- 💼 Карьера
- 🏥 Здоровье
- 🍀 Счастливое число и цвет дня

**Как использовать:**
```
/horoscope
[Выберите знак из кнопок]
```

## 🆕 Обновленные команды

### Все команды Oracle Bot:

#### 🔮 Гадание:
- `/ask` - Задать вопрос Оракулу (И-Цзин + Таро + Хорарная астр.)
- `/ritual` - Получить психологический ритуал

#### 🌟 Астрология:
- `/natal` - Натальная карта
- `/horoscope` - Гороскоп на сегодня

#### 🔢 Нумерология и матрицы:
- `/numerology` - Китайская нумерология Сюцай
- `/matrix` - Матрица Судьбы

#### ℹ️ Информация:
- `/start` - Начать работу
- `/help` - Помощь
- `/stats` - Статистика
